Nueva Documentación REST

Esta sección corresponde a la nueva versión del Servicio REST de Tiempo Aire. Incluye mejoras de seguridad, nuevos métodos, mayor estabilidad operativa y un esquema actualizado de autenticación mediante tokens.

Introducción

Los canales externos que requieran integrarse con la plataforma de Tiempo Aire pueden hacerlo mediante el Servicio Web REST.

Este servicio funge como la interfaz principal para la ejecución de operaciones transaccionales, tales como consultas, ventas, validaciones y administración de información relacionada con productos.
